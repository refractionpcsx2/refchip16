RefChip16 Chip-16 Virtual Machine Emulator
Written by Refraction

Commandline:

You can launch ROM's directly from the commandline using the following

RefChip16 -r romname.c16


Controls:

Up = Up Arrow
Down = Down Arrow
Left = Left Arrow
Right = Right Arrow
Button A = Z
Button B = X
Start = Return
Select = Right Shift


Player 2 (not that we care currently lol)

Up = W
Down = S
Left = A
Right = D
Button A = R
Button B = T
Start = Left Shift
Select = Left Control

Enjoy!
