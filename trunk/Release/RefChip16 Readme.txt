RefChip16 Chip-16 Virtual Machine Emulator
Written by Refraction

Commandline:

You can launch ROM's directly from the commandline using the following

RefChip16 -r romname.c16


Controls:

Up = Up Arrow
Down = Down Arrow
Left = Left Arrow
Right = Right Arrow
Button A = Z
Button B = X
Start = Return
Select = Right Shift

Enjoy!
